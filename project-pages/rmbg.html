PACKER

FOR FUN | WORK IN PROGRESS

Fall 2017

The idea for this app came from a complicated diagram I created while packing up my room to move out. I wrote down every container I had and then, as I packed, would note under each container what was packed in it. This quickly grew unwieldy and, by nature of being on scrap paper, temporary. I made a basic Rails app to function as a database representing all my belongings.

It was a clumsy process, having to type the names of everything: "christmas lights", "fleece", "white fan", etcetera. This brings me to the next stage: dragging and dropping items into their respective containers. I was doing some reading on Computer Vision and thought it would be neat to implement a system that removes the background from an image, selects the objects, and displays them for the user to click and drag into a container.

Currently I have the loose back-end implementation in Rails, and some JavaScript functions that work to find the items in a few of my test images.

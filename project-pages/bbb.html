FREELANCE | WORK IN PROGRESS

Fall 2017

Freelance project for my friend who is starting a book blog. It's a Ruby on Rails app hosted on Heroku. I'm currently fleshing out the back-end, adding more functionality to the editor, possibly through creating a new plug-in for slideshows or re-working an existing open source one for my purposes. I'm also including a basic CMS for the admin section of the site.

I have done minor work on the front-end, using Masonry for the Pinterest-esque tiling effect on the home page, but am looking forward to ultimately designing and updating the front end.

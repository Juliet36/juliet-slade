MINESWEEPER


After playing Minesweeper weekly for as long as I can remember, I finally decided to make it. I did so in Python, with the goal of translating it to JavaScript to embed it here.

In the process of making it, I did a lot of Minesweeper related research and learned a few things:

  • Minesweeper implements a buffer behind the scenes around the 2D matrix in order to create no special cases in checking squares.
  • The cascading algorithm is fairly simple.
  • I wanted to make it a guess-free version (I hate when there are only two squares and I guess wrong) but it may actually be less fun that way?

Blog Post with more detail on the process, the cascading algorithm, and this idea of "guess-free."

VPL

SARAH LAWRENCE COLLEGE | SUMMER RESEARCH PROGRAM

Summer 2016

Worked with professor Michael Siff and another student to create VPL (otherwise known as JAM for Juliet, Avinoam, Mike), a visual programming language. Our goal was to explore programming on a touch screen device which involved minimizing the need to type. My specific role was to implement the back-end evaluator which traversed the node-based AST we created for this language.

The language is functional and based off of scheme/lisp. It also attempts to filter the options available in the options pane at the bottom based on the type of data that could be used where the user has selected (eg. If the function returns a number, the return value must be a number as well so it only gives access to numbers, identifiers that represent numbers, and other functions that return numbers).

Towards the end of the summer it was implemented in CPS-style in orer to be able to pause the execution, and it was trampolined to deal with a stack-overflow problem. At the end of the summer I was working on making it a compiled evaluation.

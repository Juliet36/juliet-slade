SARAH LAWRENCE COLLEGE | SUMMER RESEARCH PROGRAM

Summer 2016

Summer 2016 I worked with SLC Computer Science professor Michael Siff and another student to create VPL (otherwise known as JAM for Juliet, Avinoam, Mike), a visual programming language. Our goal was to explore programming on a touch screen device which involved minimizing the need to type. My specific role was to implement the back-end evaluator which traversed the node-based AST we created for this language.

The language is functional and based off of Scheme/Lisp. It also attempts to filter the options available in the options pane at the bottom based on the type of data that could be used in the place that the user has selected. E.g. if the function returns a number, the return value must always be a number so the panel only gives access to numbers, identifiers that represent numbers, and other functions that return numbers.

The evaluator was ultimately implemented in CPS-style in order to be able to pause the execution, and it was trampolined to deal with a stack-overflow problem. At the end of the summer I was working on making it a compiled evaluation.

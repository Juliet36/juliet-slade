JS-HELP

SARAH LAWRENCE COLLEGE | INDEPENDENT STUDY IN WEB TOOLS FOR LEARNING TO PROGRAM

Spring 2017

JS-help (JS-hint already exists and the JS doubles for JavaScript and Juliet Slade!) is a tool to help students get hints while working on programming problems. I worked as a lab assistant and tutor for the Intro. to Web Programming class and observed that students often need many examples as well as help moving forward in solving problems. JS-help, given a solution, breaks it into a sequence of automatically generated hints that can be expanded much like how code folding works, but in reverse.

My demo demonstrates a potential use in which a series of problems are given from chosen categories (e.g. while loops, for loops, if/elses) and as the user works through the problems they can expand the solution code if they desire and transfer it to the editor panel. When they are ready to submit, their code is run on some unit tests and if it's correct they can move on. Depending on the number of hints they required to complete the problem, the next will be of the same level, easier, or harder.
